This is an intial project to allow entering, showing, and tracking the Grand Duchy of Ebonmarch Weapon Master results.  
A second Tournament was added to track all Valkyrie Weapon Master Results.  

The intention is to make this more visible for everyone involved to help increase and allow more interactions with these types of tournaments.  Allowing everyone to see where they started and who they have interacted with

This also includes access to the jsork to ensure that the correct name/spelling is correct to ensure accurate tracking.  Preventing duplicate names such as Waldodorf/waldorf/walddor ect...
